# Métriques Grafana - Veza

## Vue d'ensemble

Ce document décrit les métriques affichées dans Grafana.

## Métriques Système

### CPU
- Utilisation par core
- Load average
- Temps d'attente

### Mémoire
- Utilisation RAM
- Swap
- Cache

### Réseau
- Trafic entrant/sortant
- Connexions actives
- Latence

## Métriques Application

### API
- Requêtes par seconde
- Temps de réponse
- Taux d'erreur

### Base de Données
- Connexions actives
- Requêtes lentes
- Taille des tables
